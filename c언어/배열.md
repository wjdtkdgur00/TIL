<h1>배열 관련 내용</h1>

```
	1. 연속된 값의 메모리를 저장
	2. 시작주소만 알면 어디든 접근할수 있다, 배열의 이름은 시작주소이다.
	3. index = 배열 원소의 번호, int socre[5]에서 5가 index 이다.
	 3-1 ex = int scores[60] 60개의 int형 값을 가지는 배열 score
	4. char name[50] = 문자열을 저장한다.
	5.주의 할 점 : 배열의 크기를 지정하여야한다, 배열의 크기를 변수로 할 수   없다, 배열의 크기가 음수이거나 실수이면 안된다, 배열의 크기에는 변수가 올 수 없다.
	6.보통 배열을 선언할 때는 배열의 크기를 #define 지시자로 만들어진 기호 상수로 지정한다. 
	7.index값이 배열이름 뒤에 있기에 변수와 조금 다르다
	8.첨자안에 첨자가 들어갈수 있다 ex) scores[index[3]] index[3]의 값이 8이면 scores[8]이 된다.
	9.학생 5명의 성적을 입력받아 평균을 구하는 프로그램

	int scores[SIZE], i,sum=0;
	for (i = 0; i < 5; i++) {
		scanf("%d", &scores[i]);
		sum += scores[i];

	}
    printf("%.2f\n", (double)sum / 5.0);

	10. 배열을 초기화 할땐 중괄호를 사용한다. ex) int scores[5]={10,20,30,40 50}
	10-1. int scores[5]={10,20,30}이라면 나머지 두개는 0이 저장된다.
	11. 배열을 선언할때 초기화하지 않으면 배열에는 쓰레기 값이 저장된다.
	12. 배열 초기화를 한값으로 하고싶어도 {10}처럼 초기화하면 첫번째만 10이되고 나머지는 0이된다.
	13. 배열을 초기화할 때를 제외하고는 중괄호로 값을 묶어서 대입할 수 없다.
	14. 배열의 전체값을 배열원소하나의 값으로 나누면 배열원소의 값을 계산 할 수있다.
