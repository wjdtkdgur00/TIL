3. 배열을 이용하여 입력된 10개의 데이터의 최솟값과 최댓값을 출력해주는 코드
```
	srand((unsigned)time(NULL));
	int i,min=100000,max=-999999;
	int arr[10];
	for (i = 0; i <= 9; i++) {
		arr[i] = rand() % 100;
		printf("%d\n", arr[i]);
		if (min > arr[i]) {
			min = arr[i];
		}
		if (max < arr[i])
			max = arr[i];
	}
	printf("최솟값 : %d, 최댓값 : %d", min, max);

단 이 경우에는 헤더파일에 <stdlib.h>와 <time.h>를 포함해 주어야 한다.
