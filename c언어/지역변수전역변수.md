# 지역변수와 전역변수의 특징
```
	//변수의 속성에는 지금까지 배운 이름,타입,크기 외에도 범위,생존 시간, 연결등이 있다.
	//변수의 범위는 전역 변수와 지역 변수로 나뉜다.
	//전역 변수를 사용하여 디버깅할때 오류를 찾기 어렵다.(그렇기에 잘 사용하지 않는다.)
	//지역변수는 블록안에 선언되었을때 그 블록을 떠나면 안된다.
	//최신 버전의 C에서는 블록 안의 어떤 위치에서도 선언 가능하다.
	//지역 변수의 블록만 다르면 이름은 같아도 상관없다.
	//지역 변수의 생존 시간은 그 블록의 시작 ~ 끝이다.
	//함수의 헤더 부분에 정의되어 있는 매개 변수도 일종의 지역 변수이다.
	//지역 변수와 다른 점은 함수 호출시의 인수 값으로 초기화 되어 있다는 점이다.
	//전역 변수의 초기 값은 0이다.
    //어렵다
```
